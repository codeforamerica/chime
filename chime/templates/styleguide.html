Column

Defines vertically stacked items. Flexible column items can be used to fill the remaining space in a column. Columns are pretty much unnecessary unless using a flexible column item. Flexible columns only work on modern browsers (that support the flexbox model). IE8 and IE9 simply treat column items as basic block elements that do not flex.
